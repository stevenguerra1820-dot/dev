# Guía de Contribución

## 🌟 ¡Gracias por tu interés!

Agradecemos tu ayuda para mejorar este proyecto. Por favor, sigue estas guías.

## 📋 Proceso

### 1. Reportar Issues
- Usa el template de issue
- Describe claramente el problema
- Incluye pasos para reproducir
- Menciona versión y entorno

### 2. Crear Pull Requests
- Fork el repositorio
- Crea branch descriptiva
- Sigue estándares de código
- Incluye pruebas
- Actualiza documentación

### 3. Revisión de Código
- Todos los PRs requieren revisión
- El CI debe pasar
- Mantener conversaciones respetuosas
